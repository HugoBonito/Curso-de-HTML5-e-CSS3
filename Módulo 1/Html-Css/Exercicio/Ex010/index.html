Usar Links Externos

Para transformar texto em um link temos de usar a tag <a>(âncora) no texto que quero usar para o link.

Você pode usar o meu repositório público no GitHub.

Você também pode acessar o meu canal gratuito do Youtube

Quando criamos links podemos adicionar o target que serve para criar uma nova janela (target="_blank") ou então para ser na própria janela aberta (target="_self") , quando o leitor clica no link em questão, por norma o _self é o default.

Ao adicionar o rel em um link damos a entender ao Google que o link ou é para continuar para o documento atual((rel="next") ou prev), chamamos a isso link interno ou atual, ou então para uma página externa (rel="external"), que chamamos link externo.
