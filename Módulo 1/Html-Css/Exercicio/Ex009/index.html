Trabalhar com Listas

Listas Ordenadas

Para criar listas simples é necessário usar duas tags <ol> para organizar como lista e depois <li> para simbolizar cada item da lista.

A tag <ol> tens opções para mudar o tipo e em que número começa.

  1. Acordar
  2. Ligar João
  3. Tomar Café
  4. Escovar Dentes
  5. Almoçar
  6. Estudar

Listas não Ordenadas

Listas não ordenadas são usadas para por exemplo lista de compras, para isso usa se a tag <ul>, que também tem vários tipos diferentes.

  • Pão
  • Leite
  • Tomate
  • Arroz
  • Feijão

Lista de Defenição

Para criar uma Lista de Defenição(Dicionário) usa se a tag <dl> para fazer a organização da lista, para cada termo usa se a tag <dt> e por fim para a descrição de cada termo usa-se a tag <dd>.
